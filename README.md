## Run Locally

Clone the project

```bash
  git clone https://github.com/Mars-Zero/Sorting-Visualizer
```

Go to the project directory

```bash
  cd Sorting-Visualizer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Alternatively, open with Visual Studio Code and use the LiveServer extension
